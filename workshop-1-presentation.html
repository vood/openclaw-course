💓 The Heartbeat

A cron job for your agent — runs on a schedule, no prompt needed

What it does

  • ⏰ Runs every N minutes automatically
  • 📋 Executes tasks you define in HEARTBEAT.md
  • 🔄 Agent acts without being asked

Example tasks

  • 📧 Check email for new leads
  • 📊 Send daily revenue report
  • 🔔 Monitor website uptime
  • 🧹 Clean up old files

Without heartbeat → agent only responds when you message it.
With heartbeat → agent works proactively, like a real employee.

⚠️ The Risks

What can go wrong & how we handle it

🛡️ The Ground Rules

Treat it like a new employee.

  • 📂 Its own workspace, email, calendar
  • 🚫 Keep away from sensitive accounts
  • 🔍 Vet plugins before installing
  • 🎯 Be specific — vague = dangerous

💥 Internal Risks (The Real Danger)

We've seen all of these first-hand:

  • 📨 Emailed 1,000 customers without preview
  • 🗑️ Permanently deleted files while "fixing" a bug
  • 💳 Refunded Stripe charges without confirmation

Powerful agent + vague instructions = damage

🌐 External Risks

  • 🧩 ~20% of plugins are malicious
  • 🌍 390k+ instances exposed on the internet
  • 📡 Misconfiguration = open to your network
  • 🐛 Known bug allows remote hijacking

We'll set up safe defaults that limit the blast radius.
