🚨 The Anthropic Ban

What happened on April 4, 2026

🚨 What Changed

Before (worked)

  • Claude Pro ($20/mo) or Max ($200/mo)
  • Point OpenClaw at Claude via OAuth
  • Flat-rate, predictable cost
  • Everything just worked

After April 4 (blocked)

  • Subscriptions can't be used with third-party tools
  • Must use API with pay-as-you-go
  • Costs up to 50x higher
  • Thousands of agents stopped working
Platform risk is real. When you build on someone else's platform, they can change the rules overnight.

🚨 The Timeline

WhenWhat
Nov 2025OpenClaw (Clawdbot) launches, goes viral

Jan 2026Security audit: 512 vulnerabilities, 8 critical
Feb 2026Peter Steinberger (creator) joins OpenAI
Apr 4, 2026Anthropic officially blocks third-party framework usage

Why: OpenClaw users consumed 10-50x more tokens than normal chat users. Anthropic was losing money.

🔍 Alternatives

The landscape beyond OpenClaw

🔍 Claude Cowork (by Anthropic)

Anthropic's own agent. Desktop-focused, not messaging-focused.

What it does

  • Desktop agent — files, apps, browser
  • Computer use — clicks, scrolls, navigates
  • Dispatch — assign from iPhone
  • Included in Claude Pro/Max

OpenClaw vs Cowork

  • OpenClaw = always-on messaging
  • Cowork = desktop automation
  • They're complementary, not competing
  • You might want both
